# Connect Db using python script
## Install mysql connector module
```
pip3 install mysql-connector-python
```
 - create 'db.py' file

### Run Script
```
python3 db.py
```
### Commands 
- Connect db manually 
```
mysql -h <db_endpoint> -u admin -p 
```
- Show Db
```
show databases;
```
- Select DB
```
use db_name;
```
- show tables
```
show tables;
```
